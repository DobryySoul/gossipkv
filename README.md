# Gossip-KV: Встраиваемая, децентрализованная Key-Value база данных на Go

Gossip-KV — это **встраиваемая** Go-библиотека, которая позволяет создавать **децентрализованные** кластеры для обмена простой информацией (key-value). Она использует **Gossip-протокол** для синхронизации данных между узлами без внешних зависимостей.

### Зачем это нужно?

Во многих распределённых системах возникает потребность в обмене простой информацией, такой как конфигурация, сервис-дискавери или динамические флаги функций. Часто для этого приходится разворачивать и поддерживать сложные внешние сервисы вроде Redis или etcd.

**Gossip-KV** решает эту проблему, предоставляя лёгкий, самодостаточный и отказоустойчивый механизм прямо внутри вашего Go-приложения.

### Ключевые особенности

- **Встраиваемость (Embedded)**: Просто библиотека, а не отдельный сервер. Вы импортируете её и используете внутри вашего приложения.
- **Нулевая конфигурация (Zero-config)**: Узлы автоматически находят друг друга в локальной сети (используя mDNS) или через список "seed" узлов.
- **Децентрализация**: Нет главного узла (no master). Все узлы равны. Выход из строя любого узла не ломает кластер.
- **Итоговая согласованность (Eventual consistency)**: Данные со временем распространяются по всему кластеру. Это идеально для информации, которая может быть неактуальна на доли секунды.
- **Простой API**: Интуитивно понятный интерфейс `Set()` и `Get()`.

### Быстрый старт

1.  **Установка**

    ```bash
    go get [github.com/your-username/gossip-kv/pkg/gossipkv](https://github.com/your-username/gossip-kv/pkg/gossipkv)
    ```

2.  **Использование**

    Просто инициализируйте новую базу данных и начните работать с ней.

    ```go
    package main

    import (
        "fmt"
        "[github.com/DobryySoul/gossip-kv/pkg/gossipkv](https://github.com/DobryySoul/gossip-kv/pkg/gossipkv)"
    )

    func main() {
        // Создаём новый узел, который автоматически найдёт соседей.
        db, err := gossipkv.New()
        if err != nil {
            // обработка ошибки
        }

        // Установка значения
        err = db.Set("version", "v1.0.1")
        if err != nil {
            // обработка ошибки
        }

        // Получение значения
        value, err := db.Get("version")
        if err != nil {
            // обработка ошибки
        }
        
        fmt.Println("Значение ключа 'version':", value)
    }
    ```
